# SmartSpend User Stories

1. As a student shopper, I want to add an item to my shopping list so that I can keep track of what I need to buy.
2. As a student shopper, I want to edit an item in my shopping list so that I can update mistakes or quantities.
3. As a student shopper, I want to delete an item from my shopping list so that I can remove things I no longer need.
4. As a student shopper, I want to mark an item as completed so that I know it has been purchased.
5. As a student shopper, I want completed items to move to a pantry/completed state so that I do not buy them twice.
6. As a student shopper, I want to see the price of each item across multiple stores so that I can compare options easily.
7. As a student shopper, I want the cheapest store for an item highlighted so that I can save money quickly.
8. As a student shopper, I want to see the total basket cost per store so that I can decide where to shop.
9. As a student shopper, I want to set a weekly grocery budget so that I can stay within my spending limit.
10. As a student shopper, I want to see my current basket total so that I know how much I am planning to spend.
11. As a student shopper, I want a warning when I exceed my budget so that I can remove or swap items.
12. As a student shopper, I want smarter value suggestions so that I can choose cheaper purchase options.
13. As a student shopper, I want a frugal briefing summary so that I can quickly understand the cheapest way to buy my list.
14. As a student shopper, I want to see historical price information so that I can tell whether the current price is good.
15. As a student shopper, I want the app to estimate whether an item may go on sale soon so that I can decide whether to wait.
16. As a student shopper, I want to filter items by category so that I can organise my list more easily.
17. As a student shopper, I want to set a preferred store so that the app can tailor recommendations.
18. As a student shopper, I want my preferences saved so that I do not need to re-enter them every time.
19. As a developer, I want shopping and pricing data stored in a database so that the application has persistence.
20. As a developer, I want unit tests for pricing and budget calculations so that the smart logic remains correct.
21. As a student shopper, I want to create an account so that my shopping lists, budget, and preferences are saved securely.
22. As a student shopper, I want to sign in to my account so that I can access my saved shopping data later.
