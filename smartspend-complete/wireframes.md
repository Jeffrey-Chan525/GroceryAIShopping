# Low-Fidelity Wireframes

## 1. Dashboard / Home Screen
- App title at top
- Quick add item input and Add button
- Weekly budget box with progress bar
- Current basket total box
- Frugal briefing panel
- Buttons for Shopping List, Price Trends, Settings

## 2. Shopping List Screen
- Title bar and Home button
- Search/add item area
- TableView with columns:
  - Item
  - Qty
  - Coles
  - Woolworths
  - Aldi
  - Cheapest
  - Status
  - Edit/Delete
- Basket totals per store
- Compare button

## 3. Price Trends Screen
- Item selection dropdown
- Line chart area
- Current price box
- Average price box
- Trend note panel

## 4. Settings Screen
- Weekly budget input
- Preferred store dropdown
- Toggle recommendation settings
- Save settings button
- Reset data button
