# UI Template Notes

## Dashboard
Purpose: Show the main budget overview, quick-add flow, smart briefing, and real product search/import.

## Shopping List / Price Matrix
Purpose: Template for comparing item prices across stores and later displaying cheapest basket logic.

## Settings
Purpose: Template for weekly budget, preferred store, and recommendation toggles.

## Price Trends
Purpose: Template screen for later visualising `price_history` data and sale trend notes.
